Zecrypt Extension Test Page

This page contains forms to test the Zecrypt autofill extension. Make sure the extension is loaded and you're logged in.

Credit Card Form

Email Form

Instructions:

  1. Load the extension in Chrome by going to chrome://extensions/
  2. Enable "Developer mode" in the top right
  3. Click "Load unpacked" and select the Extensions folder
  4. Open this HTML file in your browser
  5. You should see "Zecrypt" buttons next to the card number and email inputs
  6. Click the extension icon to open the popup and test login functionality
